Solve two sum problem with optimal approach and give me the time complexity
🧠
Great problem! The optimal approach uses a HashMap for O(n) time complexity instead of the naive O(n²) brute force.

Here's the clean solution:
def twoSum(nums, target):
    # hashmap: value → index
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i

⏱ Time: O(n) · 💾 Space: O(n)
