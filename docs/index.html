How do I solve Two Sum optimally?
🧠
Use a HashMap for O(n) time.

For each number, check if target - num exists in the map.
def twoSum(nums, target): seen = {} for i, n in enumerate(nums): diff = target - n if diff in seen: return [seen[diff], i] seen[n] = i
⏱ O(n) time · 🗂 O(n) space
