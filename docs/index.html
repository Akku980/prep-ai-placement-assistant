Solve two sum with optimal O(n) time
🧠
The optimal approach uses a HashMap for O(n) time.
def twoSum(nums, target):
  seen = {} # val → index
  for i, n in enumerate(nums):
    if target-n in seen: return [seen[target-n], i]
    seen[n] = i
⏱ O(n) time · 💾 O(n) space

Get Started

Run it in 3 minutes

Requires Node.js, Python 3.10+, MongoDB, and a free Groq key

1
Get a free Groq API key
Sign up at console.groq.com — no credit card, 14k tokens/min free on Llama3-8B
2
Clone and configure
Clone the repo and set up your environment file
git clone https://github.com/Akku980/prep-ai-placement-assistant
cd prep-ai-placement-assistant/backend
cp .env.example .env # add your GROQ_API_KEY and MONGODB_URL
3
Start backend + frontend
# Terminal 1 — Backend
cd backend && pip install -r requirements.txt
uvicorn main:app --reload

# Terminal 2 — Frontend
cd frontend && npm install && npm run dev
Open http://localhost:5173 — sign up and start chatting
🐳
Or use Docker (easiest)
cp backend/.env.example backend/.env # fill in your keys
docker-compose up --build
App at http://localhost · API at http://localhost:8000/docs
